代码编辑工具

语法：
<insert path="文件名称.py" line=行号>
// 插入的内容
</insert>
该语法在文件的指定行插入代码。行号从1开始计数。行号为原始代码所标注的行号。对应行号原本的内容将出现在插入内容之后，例如line=6时，你的内容会出现在第6行，原本第6行的内容会后移。插入操作是注意缩进，使用标准的4空格缩进。你的行号必须出现在原本的文件中，或者原本的最大行号+1，超出范围的行号将被忽略。你的行号为原本的最大行号+1时代码将插入在原本文件的末尾。你只能操作存在于代码空间中的文件。

<delete path="文件名称.py" line=(开始行号,结束行号)>
该语法删除文件的指定行。行号从1开始计数。行号为原始代码所标注的行号。删除的内容包括开始行号和结束行号。开始行号与结束行号相同时只删除对应的一行。你只能操作存在于代码空间中的文件。

严格遵守语法规范，注意空格和引号的使用。

你不能使用插入工具多次对同一文件插入相同行号，如果需要，请在一次插入中合并所有内容。

使用示例：
<insert path="main.py" line=1>
import os

</insert>

<insert path="main.py" line=5>
print("Hello, world!")
print("1+1=",1+1)
</insert>

<delete path="main.py" line=(1,1)>
<delete path="main.py" line=(5,10)>
